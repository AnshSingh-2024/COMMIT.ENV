# 🌱 HomeHarvest AI - From Garden to Table, Powered by AI

A modern, responsive website for HomeHarvest AI - an AI-powered platform that combines sustainable living with artificial intelligence to help users grow their own food, cook delicious meals, and reduce their environmental footprint.

## 🚀 Features

### 🏠 **Home Section**
- Hero section with compelling headline and call-to-action
- Smooth gradient animations
- Responsive design for all devices

### 👨‍🍳 **My Kitchen (AI Chef Module)**
- **Upload Pantry**: Photo upload with AI ingredient recognition
- **Recipe Suggestions**: Personalized recipe recommendations based on available ingredients
- **Add Missing Ingredients**: Smart suggestions for recipe completion
- Interactive file upload with preview functionality

### 🌿 **Smart Garden**
- **Plant Diagnosis**: Upload plant photos for health assessment
- **Pest & Disease Detection**: AI-powered identification of plant issues
- **Care Reminders**: Automated scheduling for watering, fertilizing, and pruning
- Real-time plant health monitoring

### 👥 **Community Hub**
- **Upvote Recipes**: Rate and discover community-created recipes
- **Share Tips**: Exchange gardening and cooking wisdom
- **Earn Badges**: Gamified achievements for sustainable living milestones
- Social features for community engagement

### ℹ️ **About Section**
- Mission statement and project overview
- AI technology highlights
- Call-to-action for community participation

## 🛠️ Technology Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styling with TailwindCSS
- **JavaScript (ES6+)**: Interactive functionality and animations
- **TailwindCSS**: Utility-first CSS framework
- **Responsive Design**: Mobile-first approach

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. The website will load with all functionality ready to use

### Development Setup
If you want to modify the project:

1. **HTML Structure**: Edit `index.html` for content changes
2. **Styling**: Modify `style.css` for custom CSS
3. **Functionality**: Update `script.js` for JavaScript features
4. **TailwindCSS**: The project uses CDN version, no build process required

## 🔧 Customization

### Adding New Sections
1. Add new section to `index.html`
2. Update navigation links
3. Add corresponding styles in `style.css`
4. Include any JavaScript functionality in `script.js`

### Modifying Colors
Update the TailwindCSS configuration in `index.html`:
```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                'green': {
                    // Your custom green shades
                }
            }
        }
    }
}
```
